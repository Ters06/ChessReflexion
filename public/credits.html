Crédits et Remerciements

Cette application n'aurait pas pu voir le jour sans le travail exceptionnel de la communauté open-source et des services tiers formidables.

  • chess.js

    Pour la validation des coups et toute la logique du jeu d'échecs.

  • Tailwind CSS

    Pour le framework CSS qui a permis de créer ce design.

  • Google Fonts

    Pour la police d'écriture "Inter".

  • Netlify

    Pour l'hébergement et les fonctions serverless.

  • Neon

    Pour la base de données PostgreSQL serverless.
