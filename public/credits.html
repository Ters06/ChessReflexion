Crédits et Remerciements

Cette application n'aurait pas pu voir le jour sans le travail remarquable de nombreux développeurs et communautés. Nous tenons à les remercier chaleureusement.

Technologies et Librairies

  • Hébergement et Backend : Netlify pour sa plateforme JAMstack révolutionnaire.
  • Base de Données : Neon pour sa base de données PostgreSQL serverless.
  • Logique Échiquéenne : La librairie chess.js par Jeff Hlywa, le cerveau derrière la validation des coups.
  • Framework CSS : Tailwind CSS pour son approche "utility-first" qui a permis de designer rapidement cette interface.
  • Authentification : Les services d'authentification robustes de Google Cloud Platform.

Inspiration et Connaissances

Un grand merci à la communauté échiquéenne mondiale pour le partage constant de connaissances, et notamment :

  • La plateforme Lichess.org pour son engagement envers les échecs open-source.
  • Les nombreux auteurs et coachs dont les méthodes de réflexion ont inspiré cet outil.
